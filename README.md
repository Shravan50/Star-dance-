# 💻 ShravanOS

Welcome to **ShravanOS**! This is a browser-based, windowed operating system built to advance my front-end development skills in JavaScript, CSS, and HTML. 

The goal of this project is to master DOM manipulation, event handling, and custom asynchronous UI components within a clean, interactive environment.

---

## 🚀 Live Features

* **Custom Window Manager:** Fully draggable desktop windows with functional custom Open/Close states.
* **Live Webcam App:** Real-time user video streaming utilizing native browser media APIs.
* **Interactive Calculator:** A fully operational grid-based calculator capable of handling real-time math operations safely.
* **Notes App:** A clean, workspace-integrated text area for quick scratchpad notation.
* **Mock Browser:** A stylized dashboard utility mirroring a standard search engine interface.
* **System Personalization:** Dynamic custom greeting and system state adaptation driven by user-inputted profile metadata.
* **Drawing Pad** A way to showcase your drawing skills!!

---

## 🛠️ Built With

* **HTML5:** Semantic document structuring and layout configuration.
* **CSS3:** Custom absolute positioning, layout grids, backdrop blurs, and interface styling.
* **Vanilla JavaScript:** Event listeners, asynchronous media stream ingestion, calculator logic parsing, and programmatic element tracking.
